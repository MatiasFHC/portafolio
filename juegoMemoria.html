Juego de Memoria en C#

Este proyecto personal fue creado para ofrecer una experiencia entretenida. El juego desafía a los usuarios a encontrar pares de imágenes iguales, utilizando lógica de programación para manejar las interacciones y las reglas del juego.

Características principales:

  • Interfaz amigable con diseño intuitivo
  • Lógica de juego basada en eventos, asegurando una experiencia fluida
  • Ideal para mejorar la memoria y el enfoque
