El juego del Molino (1er Ciclo)

Este proyecto recrea el clásico juego de estrategia, donde dos jugadores compiten para formar líneas de tres fichas consecutivas (molinos) y eliminar las piezas del oponente. Desarrollado con una interfaz gráfica intuitiva y lógica de juego para permitir que el juego se desarrolle correctamente.

Características principales:

  • Interfaz gráfica: Diseño claro y fácil de usar que facilita la interacción de los jugadores.
  • Lógica de juego estratégica: Implementación de reglas tradicionales del juego, con manejo de turnos y validación de movimientos.
  • Eliminación de piezas: Mecánica para retirar fichas del oponente tras formar un molino.
