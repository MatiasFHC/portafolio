Visualización de Estadísticas del Covid-19 en Perú (2do Ciclo)

Este programa facilita el análisis de datos relacionados con el Covid-19 en Perú mediante la generación de gráficos y tablas informativas. Utiliza bibliotecas de Python para procesar y visualizar información clave, lo que permite a los usuarios explorar datos de manera clara y comprensible.

Características principales:

  • Gráficos dinámicos: Representaciones visuales como líneas, barras y sectores para mostrar tendencias y proporciones.
  • Tablas interactivas: Resumen detallado de estadísticas, como casos confirmados, recuperados y fallecimientos.
  • Bibliotecas utilizadas: Matplotlib y Pandas para análisis y visualización.
